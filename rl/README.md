# Laboratory on Reinforcement Learning

This repository contains material and code for a laboratory session on Machine Learning with Python, focusing on **Reinforcement Learning**.

## Overview

The notebook guides you through:
- Basic introduction to the Gym library  
- SARSA
- Q-learning
- Deep Q-learning with memory buffer

---

## Google Colaboratory

If using Google Colab, these libraries are typically preinstalled.

All the lab materials and datasets are available at the following link: [Google Drive Folder](https://drive.google.com/drive/folders/1P8QnqSo5b0QjzHI9IjMraPJCDkrjXWFA?usp=drive_link)

To open the notebook in Google Colab:

1. Right-click on `main.ipynb`
2. Select **Open with**
3. Choose **Google Colaboratory**
