# Laboratory on PyTorch and Deep Learning

This repository contains material and code for a laboratory session on *Deep Learning with Python*, focusing on *PyTorch* and fundamental neural-network architectures.

Link to the [Notebook](https://colab.research.google.com/drive/1_nROB9287OISaHvj5OR-sr16I-BrUKIx).

## Overview

The notebook guides you through:

* Introduction to PyTorch
* Working with Google Colab (GPU runtime, environment setup)
* Loading and understanding a dataset
* Building and training neural-network models:

  * Multi-Layer Perceptrons (MLP)
  * Convolutional Neural Networks (CNN)

* Core deep-learning components:

  * Loss functions
  * Backpropagation
  * Optimization

* Optimizing hyperparameters
* Tracing and analyzing learning curves

## Prerequisites

### Requirements

If you want to run the notebook on your machine make sure you have the following installed:

* Python >= 3.7
* [PyTorch](https://pytorch.org/get-started/locally/)
* [NumPy](https://numpy.org/install/)
* [Matplotlib](https://matplotlib.org/stable/users/installing.html)
* [Torchvision](https://pytorch.org/vision/stable/)
* [Pillow](https://python-pillow.org/)
* [Optuna](https://optuna.org/)
* [Tensorboard](https://www.tensorflow.org/tensorboard)
* [wandb](https://wandb.ai/)

Optional (for experimentation):

* GPU-enabled CUDA toolkit (if running locally)
