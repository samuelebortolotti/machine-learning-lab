# Laboratory on Unupervised Learning

This repository contains material and code for a laboratory session on Machine Learning with Python, focusing on **Scikit-learn** and **unsupervised learning**.  

## Google Colaboratory

Please find the lab material at the following link: [Google Drive Folder](https://drive.google.com/drive/folders/1kAcXAs8NF1dMd1SMeWG_UJqF_HWQL7_1?usp=drive_link)

To open the notebook in Google Colab:

1. Right-click on `unsupervised-lab.ipynb`
2. Select **Open with**
3. Choose **Google Colaboratory**
